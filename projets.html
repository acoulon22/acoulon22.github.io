Projet Calcul facture pour gîte
Projet 3 (personnel)

Description : Ce projet, écrit en langage C, a été réalisé personnellement pour facilité les calculs pour une location d'un gîte (de mes proches).
La complexité n'était pas vraiment le fait de calculer l'acompte d'une location ou le montant global, mais plutôt le calcul de la taxe de séjour qui n'est pas simple à faire manuellement.
L'affichage de ce programme est complet, il affiche le montant global, l'acompte, la taxe de séjour ainsi que le montant de la location après avoir régler l'acompte.

Instruction pour accèder au projet : une fois arrivé sur la démo, cliquez sur "Fork this", choisissez le langage "C", puis appuyez sur "Run".
PS : Il ne faut surtout pas que le mode "Google Traduction" ou autre traducteur soit activé sur la page web du site onlinegdb.com, cela ferait planter le compilateur.

Projet Calcul électricité
Projet 4 (personnel)

Description : Ce projet a été réalisé personnellement pour facilité les calculs d'électricité d'une location saisonnière d'un gîte (de mes proches).
Ce programme, écrit en langage C, ne se soucie pas de la location du compteur électrique, ce qui le rend efficace. De plus, il prend en compte toutes les taxes et celles-ci ont été préalablement initialisées en constante pour prévoir un potentiel changement du taux de taxe ou bien du prix de l'électricité.

Instruction pour accèder au projet : une fois arrivé sur la démo, cliquez sur "Fork this", choisissez le langage "C", puis appuyez sur "Run".

Projet Calcul nombre de jours de dates
Projet 5 (personnel)

Description : Ce projet a été réalisé personnellement suite à la demande de l'un mes proches.
Ce programme, écrit en langage Java, est décomposé en 3 actions possibles : saisir une date pour avoir le nombre de jour(s) de celle-ci dans l'année donnée, saisir deux dates pour savoir combien de jour(s) les séparent et pour finir, il y a évidemment l'action pour quitter le programme.

Instruction pour accèder au projet : une fois arrivé sur la démo, cliquez sur "Fork this", choisissez le langage "Java", puis appuyez sur "Run".
